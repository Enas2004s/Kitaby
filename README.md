# Kitaby Platform

## Project Idea
Kitaby is a platform designed to help students share and request books easily.  
Users can upload books, browse categories, and send requests for available books.

---

## Features
- User registration and login
- Add and manage books
- Book request system (approve / reject)
- Categories system
- Admin dashboard with real statistics

---

## Admin Dashboard
The platform includes a dashboard for admin users only.  
It displays real statistics such as:
- Number of users
- Number of books
- Number of categories
- Number of requests

---

## Technologies Used
- Laravel (PHP)
- MySQL
- HTML / CSS
- AdminLTE (Dashboard Template)

---

## How to Run the Project
1. Install XAMPP
2. Import the database
3. Run:
   php artisan migrate
   php artisan serve
4. Open:
   http://127.0.0.1:8000
